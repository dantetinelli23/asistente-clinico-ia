# Asistente Clínico IA

Sistema de IA en producción para apoyo a la decisión clínica: RAG sobre
protocolos médicos, agente cíclico con evaluación y escalamiento a humano,
consulta segura de historias clínicas, observabilidad y despliegue serverless.

**Stack:** Python · LangChain · LangGraph · FastAPI · Docker

**Estado:** en construcción 🚧
